# API Tester Lite

A lightweight, focused Postman-like API testing tool. Send HTTP requests and view responses with ease.

## Features

- One-click API testing
- Support for all HTTP methods (GET, POST, PUT, DELETE, PATCH, HEAD, OPTIONS)
- Headers and query parameters management
- Request body editor
- Authentication support (Bearer, Basic, API Key)
- Response viewer with syntax highlighting
- Response time and size tracking
- Code snippet generation (JavaScript, Python, cURL)
- Request history

## Tech Stack

- Next.js 14
- TypeScript
- Tailwind CSS
- tRPC

## Getting Started

```bash
npm install
npm run dev
```

Open http://localhost:3000 to start testing APIs.
